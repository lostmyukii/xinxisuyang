## 审计原则

## 手动正式版完成结论

第三轮正式 4 小时测试已同时证明：

- API 与 UI 通道均由 `caffeinate-ims` 保持唤醒并连续运行 240 分钟；
- 每条通道均获得 480 个有效样本，高于 456 个覆盖率门槛；
- 最大采样间隔分别为 30.49 秒和 30.10 秒，低于 60 秒门槛；
- 服务状态、501 条记录、单一快照、后台 8 行和大屏 1–12 行均符合预期；
- 瞬时请求失败均为 0；
- API、UI 服务和系统 Chrome 预热后持续内存增长均为 0 MiB，滚动峰值增长分别为 6.18、0.47 和 3.20 MiB。

因此，手动导入正式版满足《开发文档》第 17 节完成条件。完整机器可读报告位于 Git 忽略的 `logs/`，正式摘要已写入《测试与验收报告》。自动采集仍受下节边界约束，不属于本次完成声明。
